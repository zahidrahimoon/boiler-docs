## Project Structure

This explains the purpose, structure, and conventions of the **Next.js Enterprise Starter Kit**. It is intended for contributors, maintainers, and teams building production-ready Next.js applications.

### High-Level Technologies

- Next.js 15 (App Router + server components)  
- React 18  
- TypeScript  
- TailwindCSS 4  
- Storybook  
- Jest / Vitest  
- Redux Toolkit  
- React Query  
- Socket.io  
- i18n (next-intl)  
- Zod (validation)  
- Pino (logging)

#### `socket/`

Input and schema validation (Zod)

#### `public/`
Static assets

#### `__tests__/`

### Conventions and Architecture Notes

- TypeScript-first: Always type props and return types  
- Component composition preferred over large prop interfaces  
- TailwindCSS utilities; extract repeated patterns using helpers  
- Redux for global state, React Query for server-side data  
- Side effects in hooks or actions, not in components  
- Write tests: unit â†’ integration â†’ e2e
