## Next.js Enterprise Starter Kit

<div className="bg-[#0A0A0A] border border-neutral-800 text-slate-200 p-8 rounded-xl shadow-xl leading-relaxed"> 

A **production‑grade**, **scalable**, and **enterprise‑oriented** boilerplate built with **Next.js 15**, **TypeScript**, **Tailwind CSS 4**, **shadcn/ui**, **Redux Toolkit**, and **React Query**. Designed to support multilingual, real‑time, authenticated applications with a strong emphasis on maintainability, architecture, and long‑term developer experience.

This boilerplate delivers a fully modular foundation, enabling teams to build complex systems with confidence, performance consistency, and architectural clarity.

### Key Features

#### Enterprise-Ready Architecture
- Modular, domain-driven folder structure
- Full support for the **Next.js App Router**
- Component-driven development with Storybook
- Optimized bundling with lazy loading and code splitting

#### Dynamic Layout System
- Two refined layout modes: **Website** and **Dashboard**
- Fully responsive, mobile-first design
- Native RTL support (Arabic, Urdu, Persian)
- Layout state persistence via localStorage

#### Robust Authentication System
- Full auth suite: login, signup, OTP, forgot password, reset password
- Secure cookie-based session handling
- Flow-validated OTP and password reset routes
- Middleware-driven route protection at the **page level**

#### Real-Time Capabilities
- Native **Socket.IO** and WebSocket integration
- Local Express server provided for real-time testing
- Automatic reconnection, typed events, and error recovery

#### Internationalization (i18n)
- Locale-based routing using the App Router
- RTL/LTR adaptive styling baked into the system
- JSON-based translation dictionaries

#### UI System (50+ Components)
- Built with **shadcn/ui** and **Radix UI** primitives
- Theme-driven, token-based styling via Tailwind CSS 4
- Includes enterprise utilities: Data Table, Scroll-to-Top, Error Boundaries

</div>

### Why This Boilerplate?

#### Superior Developer Experience
- Strict TypeScript mode for safer production builds
- ESLint + Prettier enforced coding standards
- Clear separation of concerns across modules
- Pluggable architecture for forms, layouts, networking, and state management

#### Production-Level Performance
- React Query caching and request deduplication
- Server components for reduced bundle size
- Optimized icon and asset loading
- Virtualized data tables for heavy datasets

#### Enterprise Scalability
- Designed for multi-team development
- Extensible layout and authentication systems
- Comprehensive folder conventions for CI/CD readiness
