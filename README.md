# Next.js Enterprise Starter Kit - Documentation

Comprehensive documentation for the Next.js Enterprise Starter Kit, built with Nextra 4 and the App Router.

1. Clone the repository

   ```bash
   git clone https://github.com/zahidrahimoon/next-starter.git
   cd nextjs-starter
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Set up environment variables

   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. Run the development server

   ```bash
   npm run dev
   ```
